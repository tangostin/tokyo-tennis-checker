# tokyo-tennis-checker
## システムの運用・保守マニュアル

本システム（30分間隔の自動巡回）は、**Google Apps Script (GAS)** のタイマーを起点として動作しています。

### ⚙️ 定期実行（タイマー）の確認・編集
タイマーの変更やコードの確認を行う場合は、以下のいずれかでGASのエディタを開きます。
1. **Google ドライブ** ＞ 作成した「テニスコート自動巡回タイマー」のファイルを開く
2. **Google Apps Script トップ**（ https://script.google.com/ ）＞ プロジェクト一覧から選択

---

### 🛑 緊急停止（シャットダウン）手順
巡回を一時的に止めたい場合は、**GitHub側で窓口を閉鎖する**のが最も迅速で確実です。

#### 【方法A】GitHubで一瞬で止める（推奨：緊急時・出先からなど）
1. 本リポジトリの **[Actions]** タブを開く
2. 左メニューから本ワークフロー（ `Tennis Court Auto Checker` ）を選択
3. 画面右側にある **[⚙️ (歯車マーク)]** または **[...]** ボタンをクリック
4. **[Disable workflow]** （赤文字）をクリックして無効化する
※ 再開する時は、同じ場所に表示される **[Enable workflow]** をクリックすれば復旧します。

#### 【方法B】GAS側で完全に止める（長期間利用しない場合）
1. GASのエディタ画面左メニューから **⏰（トリガー）** を開く
2. 30分おきに設定されているトリガーの右端 **[︙]** をクリック
3. **[トリガーを削除]** を実行
※ 再開する時は、再度「＋ トリガーを追加」から分ベース（30分おき）で再設定が必要です。
